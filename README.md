# e-plantShopping: Paradise Nursery

## Project Description
Paradise Nursery is a React-based e-commerce application designed for houseplant enthusiasts. This application allows users to browse a diverse collection of air-purifying and aromatic plants, learn about their benefits, and manage a digital shopping cart.

## Features
* **Landing Page**: Features a "Get Started" button and the company tagline.
* **Product Listing**: A grid display of various plants with images, descriptions, and pricing.
* **Shopping Cart**: Real-time cart management using Redux Toolkit, allowing users to:
    * Add items to the cart.
    * Increase or decrease item quantities.
    * Remove items from the cart.
    * View the total cost calculation automatically.

## Tech Stack
* **Frontend**: React.js
* **State Management**: Redux Toolkit
* **Styling**: CSS3
* **Bundler**: Vite
